# Proworktree

Project assignment work for Proworktree

## Features

- Login User
- Register User
- Google Authentication (currently broken)
- Fetch user data from [reqres.in](https://reqres.in/)


## Packages used
- google_sign_in
- firebase_core
- firebase_auth
- lottie
- google_fonts
- modal_progress_hud
